# Cập nhật current time cho máy chủ

Ngày đăng

Nov 6, 2022 2:23 PM

Khi máy chủ của bạn bị chết một khoảng thời gian và sống lại thì đôi lúc thời gian của máy chủ sẽ bị lệnh so với thời gian thực tế, để cập nhật lại current time cho máy chủ thì các bạn làm như sau.

Copy

```
CURRENT=$(wget -qSO- --max-redirect=0 google.com 2>&1 | grep Date: | cut -d' ' -f5-8)Z
```

Copy

```
sudo date -s "$CURRENT"
```
