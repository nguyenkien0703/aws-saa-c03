## Helm

Helm là trình quản lý gói cho Kubernetes. Xem hướng dẫn cài đặt tại: [https://helm.sh/docs/intro/install/](https://helm.sh/docs/intro/install/)

## Cờ toàn cục (Global Flags)

| Cờ | Mô tả |
| --- | --- |
| `--kube-context [name]` | Tên ngữ cảnh Kubernetes để sử dụng |
| `--namespace [name]` | Tên không gian tên cho thao tác này |

## Quản lý kho lưu trữ (Repository Management)

| Lệnh | Mô tả |
| --- | --- |
| `helm repo add [name] [url]` | Thêm một kho lưu trữ |
| `helm repo list` | Liệt kê tất cả kho lưu trữ đã thêm |
| `helm repo update` | Cập nhật bộ nhớ đệm cục bộ của các chart có sẵn |
| `helm repo remove [name]` | Xóa một kho lưu trữ |
| `helm search repo` | Liệt kê tất cả chart trong các kho lưu trữ |
| `helm search repo [keyword]` | Tìm kiếm chart theo từ khóa trong kho lưu trữ |

## Cài đặt Helm Chart (Installing Helm Charts)

| Lệnh | Mô tả |
| --- | --- |
| `helm install [name] [chart]` | Cài đặt chart với tên cụ thể |
| `helm install [name] [chart] --generate-name` | Cài đặt chart và tự động tạo tên |
| `helm install [name] [chart] --namespace [namespace]` | Cài đặt chart vào không gian tên cụ thể |
| `helm install [name] [chart] --set [key]=[value]` | Cài đặt chart với giá trị tùy chỉnh |
| `helm install [name] [chart] --values [file]` | Cài đặt chart với giá trị từ tệp YAML |
| `helm install [name] [chart] --dry-run` | Mô phỏng cài đặt chart mà không thực thi |

## Quản lý bản phát hành (Release Management)

| Lệnh | Mô tả |
| --- | --- |
| `helm list` | Liệt kê tất cả bản phát hành đã cài đặt |
| `helm list --namespace [namespace]` | Liệt kê bản phát hành trong không gian tên cụ thể |
| `helm upgrade [release] [chart]` | Nâng cấp bản phát hành lên phiên bản chart mới |
| `helm upgrade [release] [chart] --set [key]=[value]` | Nâng cấp bản phát hành với giá trị tùy chỉnh |
| `helm rollback [release] [revision]` | Khôi phục bản phát hành về phiên bản trước đó |
| `helm uninstall [release]` | Gỡ cài đặt một bản phát hành |
| `helm uninstall [release] --keep-history` | Gỡ cài đặt nhưng giữ lại lịch sử bản phát hành |
| `helm history [release]` | Xem lịch sử các phiên bản của bản phát hành |
| `helm status [release]` | Xem trạng thái của bản phát hành |

## Kiểm tra và gỡ lỗi (Inspecting and Debugging)

| Lệnh | Mô tả |
| --- | --- |
| `helm lint [chart]` | Kiểm tra chart xem có lỗi cấu hình không |
| `helm show values [chart]` | Hiển thị giá trị mặc định của chart |
| `helm show chart [chart]` | Hiển thị thông tin metadata của chart |
| `helm get values [release]` | Lấy giá trị cấu hình của bản phát hành |
| `helm get manifest [release]` | Lấy manifest Kubernetes của bản phát hành |
| `helm template [name] [chart]` | Tạo manifest từ chart mà không cài đặt |

## Lệnh khác (Other Commands)

| Lệnh | Mô tả |
| --- | --- |
| `helm version` | Hiển thị phiên bản Helm hiện tại |
| `helm env` | Hiển thị các biến môi trường của Helm |
| `helm dependency list [chart]` | Liệt kê các phụ thuộc của chart |
| `helm dependency update [chart]` | Cập nhật phụ thuộc của chart |
| `helm package [chart]` | Đóng gói chart thành tệp .tgz |
