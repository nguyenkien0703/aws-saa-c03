Chia sẻ cách thực hành AWS mà không cần tạo tài khoản để tránh mất phí. Phù hợp với các bạn mới muốn học AWS. Không khuyến kích dùng cho production.

![image](https://images.spr.so/cdn-cgi/imagedelivery/j42No7y-dcokJuNgXeA0ig/fb0bf7e4-50a3-4115-84ea-0734728a413e/Untitled/w=1920,quality=90,fit=scale-down)

Để thực hiện ta sẽ giả lập môi trường AWS dưới máy bằng cách sử dụng công cụ [LocalStack](https://localstack.cloud/). Các bước thực hiện như sau.

Chạy LocalStack bằng Docker:

Copy

```
docker run --rm -it -p 4566:4566 -p 4510-4559:4510-4559 localstack/localstack
```

Cài [AWS CLI](https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html) và chạy câu lệnh `configure` và nhập như bên dưới:

Copy

```
aws configure
```

Copy

```
AWS Access Key ID: test
AWS Secret Access Key: test
Default region name: us-east-1
Default output format [None]
```

Cuối cùng khi chạy câu lệnh AWS thì ta thêm thuộc tính `--endpoint-url` vào:

Copy

```
aws --endpoint-url=http://localhost:4566
```

Ví dụ tạo S3 Bucket:

Copy

```
aws --endpoint-url=http://localhost:4566 s3api create-bucket --bucket localstack
```

Copy

```
{
	"Location": "/localstack"
}
```

Nếu muốn sử dụng UI thì các bạn làm theo hướng dẫn ở đây [**Resource Browser**](https://docs.localstack.cloud/user-guide/web-application/resource-browser/)**.** Tuy còn nhiều hạn chế nhưng đây là công cụ rất tiện dụng cho các bạn cần thực hành AWS mà không thể tạo tài khoản. LocalStack hỗ trợ giả lập hơn 60 dịch vụ AWS, tha hồ thực hành nhé.
