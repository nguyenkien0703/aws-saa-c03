Docker cung cấp một đoạn script giúp ta cài đặt Docker lên các môi trường Ubuntu, Centos, Fedora, … một cách nhanh chóng.

![image](https://images.spr.so/cdn-cgi/imagedelivery/j42No7y-dcokJuNgXeA0ig/8db993ce-2b1d-473a-bd37-4f3852072568/Kubernetes_c_bn_(2)/w=1920,quality=90,fit=scale-down)

Chạy câu lệnh sau để cài Docker:

Copy

```
curl -fsSL https://get.docker.com/ | sh
```

Câu lệnh trên tải đoạn script cài Docker xuống và thực thi, đợi đoạn script chạy xong thì Docker đã được cài vào máy. Nếu bạn muốn xem trước đoạn script sẽ chạy gì thì chạy câu lệnh sau:

Copy

```
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh ./get-docker.sh --dry-run
```

Sau khi đoạn script chạy xong ta chạy câu lệnh sau để gõ `docker` không cần `sudo`:

Copy

```
sudo usermod -aG docker $(whoami)
```
