# Xóa toàn bộ container đang ở trạng thái exited

Ngày đăng

Nov 18, 2022 3:16 PM

Khi làm việc với Docker thì ta sẽ có nhu cầu xóa hết toàn bộ container đang ở trạng thái *exited* thay vì phải gõ từng dòng với câu lệnh `docker rm <container-id>`, ta làm như sau:

Copy

```
docker rm $(docker ps -a -f status=exited -q)
```
