## Content

Danh sách các dịch vụ AWS chính, hỗ trợ quản lý tài nguyên, ứng dụng, dữ liệu và bảo mật trên đám mây.

## Dịch vụ mạng (Networking)

| Dịch vụ | Mô tả | Ứng dụng |
| --- | --- | --- |
| **VPC** (Amazon Virtual Private Cloud) | Cung cấp đám mây riêng ảo, cô lập để khởi chạy tài nguyên AWS. | Tạo mạng riêng, subnet, gateway. |
| **Direct Connect** | Kết nối mạng chuyên dụng từ mạng nội bộ đến AWS. | Truyền dữ liệu tốc độ cao, ổn định. |
| **Route 53** | Dịch vụ DNS có khả năng mở rộng và đăng ký tên miền. | Quản lý DNS, định tuyến traffic. |
| **CloudFront** | Phân phối nội dung với độ trễ thấp, tốc độ cao. | CDN cho website, video streaming. |

## Dịch vụ tính toán (Compute)

| Dịch vụ | Mô tả | Ứng dụng |
| --- | --- | --- |
| **EC2** (Elastic Compute Cloud) | Cung cấp tài nguyên tính toán có thể thay đổi kích thước. | Chạy máy ảo, ứng dụng web. |
| **Elastic Beanstalk** | Container ứng dụng để triển khai và quản lý ứng dụng. | Triển khai ứng dụng nhanh, tự động mở rộng. |
| **Lambda** | Chạy mã theo sự kiện, tự động quản lý tài nguyên. | Xử lý sự kiện, serverless app. |
| **EC2 Container Service (ECS)** | Chạy và quản lý container Docker trên cụm EC2. | Ứng dụng microservices. |
| **GameLift** | Triển khai và mở rộng game multiplayer dựa trên phiên. | Game online không chi phí ban đầu. |

## Dịch vụ lưu trữ (Storage)

| Dịch vụ | Mô tả | Ứng dụng |
| --- | --- | --- |
| **S3** (Simple Storage Service) | Lưu trữ và truy xuất dữ liệu bất kỳ dung lượng. | Lưu trữ file, backup. |
| **Elastic File System (EFS)** | Dịch vụ lưu trữ file cho EC2. | Chia sẻ file giữa các instance. |
| **Glacier** | Lưu trữ chi phí thấp, bền vững cho dữ liệu lưu trữ. | Lưu trữ dài hạn, backup. |
| **Storage Gateway** | Tích hợp môi trường tại chỗ với lưu trữ đám mây. | Backup, khôi phục thảm họa. |
| **Import/Export Snowball** | Di chuyển dữ liệu lớn vào/ra AWS bằng thiết bị vật lý. | Chuyển dữ liệu offline. |

## Dịch vụ di động (Mobile Services)

| Dịch vụ | Mô tả | Ứng dụng |
| --- | --- | --- |
| **Mobile Hub** | Xây dựng, thử nghiệm, giám sát ứng dụng di động. | Phát triển app nhanh. |
| **Device Farm** | Kiểm tra ứng dụng trên điện thoại/thiết bị thực. | Cải thiện chất lượng app. |
| **Mobile Analytics** | Thu thập, trực quan hóa dữ liệu sử dụng ứng dụng. | Phân tích hành vi người dùng. |
| **Cognito** | Quản lý danh tính và đồng bộ dữ liệu người dùng. | Đăng nhập, đồng bộ app. |

## Dịch vụ bảo mật (Security)

| Dịch vụ | Mô tả | Ứng dụng |
| --- | --- | --- |
| **IAM** (Identity and Access Management) | Kiểm soát truy cập dịch vụ và tài nguyên AWS. | Quản lý quyền người dùng. |
| **Inspector** | Phân tích ứng dụng để tìm lỗ hổng bảo mật. | Kiểm tra bảo mật app. |
| **Certificate Manager** | Quản lý và triển khai chứng chỉ SSL/TLS. | Bảo mật website. |
| **Directory Service** | Cung cấp thư mục quản lý trên đám mây. | Quản lý danh tính doanh nghiệp. |
| **WAF** (Web Application Firewall) | Bảo vệ ứng dụng web khỏi tấn công như SQL injection. | Lọc lưu lượng web. |

## Dịch vụ quản lý và giám sát (Management & Monitoring)

| Dịch vụ | Mô tả | Ứng dụng |
| --- | --- | --- |
| **CloudWatch** | Giám sát tài nguyên và ứng dụng. | Theo dõi hiệu suất, log. |
| **Config** | Lưu danh sách tài nguyên, lịch sử cấu hình. | Kiểm tra thay đổi cấu hình. |
| **CloudTrail** | Ghi lại các cuộc gọi API để tăng khả năng quan sát. | Kiểm tra hoạt động người dùng. |
| **Trusted Advisor** | Kiểm tra môi trường AWS để tối ưu chi phí, bảo mật. | Cải thiện hiệu suất hệ thống. |

## Dịch vụ DevOps (DevOps Tools)

| Dịch vụ | Mô tả | Ứng dụng |
| --- | --- | --- |
| **CloudFormation** | Tạo và cập nhật tài nguyên AWS theo mẫu. | Tự động hóa hạ tầng. |
| **OpsWorks** | Nền tảng DevOps để quản lý ứng dụng. | Quản lý ứng dụng phức tạp. |
| **CodeCommit** | Dịch vụ quản lý mã nguồn với Git. | Lưu trữ repository riêng. |
| **CodeDeploy** | Tự động hóa triển khai mã nguồn. | Triển khai ứng dụng không downtime. |
| **CodePipeline** | Dịch vụ CI/CD để phát hành phần mềm. | Tự động hóa quy trình phát triển. |
| **Service Catalog** | Quản lý danh mục tài nguyên IT được phê duyệt. | Cung cấp portal cho nhân viên. |

## Dịch vụ ứng dụng (Application Services)

| Dịch vụ | Mô tả | Ứng dụng |
| --- | --- | --- |
| **SNS** (Simple Notification Service) | Gửi thông báo đến người dùng hoặc ứng dụng. | Thông báo push, email. |
| **SQS** (Simple Queue Service) | Hàng đợi tin nhắn có khả năng mở rộng cao. | Xử lý tin nhắn bất đồng bộ. |
| **SES** (Simple Email Service) | Gửi và nhận email. | Chiến dịch email marketing. |
| **AppStream** | Phát ứng dụng nặng từ đám mây đến thiết bị. | Streaming game, phần mềm. |
| **Elastic Transcoder** | Chuyển đổi file media trên đám mây. | Chuyển mã video giá rẻ. |
| **API Gateway** | Tạo, giám sát và bảo mật API. | Xây dựng API cho ứng dụng. |
| **SWF** (Simple Workflow) | Điều phối các bước xử lý trong ứng dụng. | Quản lý quy trình công việc. |
| **CloudSearch** | Dịch vụ tìm kiếm quản lý cho website/app. | Tích hợp tìm kiếm vào ứng dụng. |

## Dịch vụ cơ sở dữ liệu (Database)

| Dịch vụ | Mô tả | Ứng dụng |
| --- | --- | --- |
| **RDS** (Relational Database Service) | Quản lý cơ sở dữ liệu quan hệ. | MySQL, PostgreSQL, Oracle. |
| **ElastiCache** | Cải thiện hiệu suất với bộ nhớ đệm. | Redis, Memcached. |
| **DynamoDB** | Cơ sở dữ liệu NoSQL có khả năng mở rộng. | Ứng dụng thời gian thực. |
| **Redshift** | Kho dữ liệu petabyte-scale, chi phí thấp. | Phân tích dữ liệu lớn. |
| **DMS** (Database Migration Service) | Di chuyển cơ sở dữ liệu lên đám mây. | Chuyển đổi DB với ít downtime. |

## Dịch vụ phân tích và dữ liệu lớn (Analytics & Big Data)

| Dịch vụ | Mô tả | Ứng dụng |
| --- | --- | --- |
| **EMR** (Elastic MapReduce) | Thực hiện tác vụ big data như phân tích log. | Hadoop, Spark. |
| **Kinesis** | Xử lý dữ liệu streaming thời gian thực. | Phân tích dữ liệu trực tiếp. |
| **Data Pipeline** | Điều phối workflow dữ liệu định kỳ. | ETL dữ liệu. |
| **Elasticsearch Service** | Triển khai và mở rộng Elasticsearch. | Tìm kiếm, phân tích log. |
| **Machine Learning** | Xây dựng ứng dụng thông minh. | Dự đoán, phân loại dữ liệu. |

## Dịch vụ IoT và doanh nghiệp (IoT & Enterprise)

| Dịch vụ | Mô tả | Ứng dụng |
| --- | --- | --- |
| **AWS IoT** | Kết nối thiết bị với ứng dụng đám mây. | IoT, nhà thông minh. |
| **WorkSpaces** | Dịch vụ máy tính để bàn trên đám mây. | Làm việc từ xa. |
| **WorkDocs** | Lưu trữ và chia sẻ tài liệu doanh nghiệp. | Hợp tác nhóm. |
| **WorkMail** | Dịch vụ email và lịch quản lý. | Email doanh nghiệp. |

---

## Mẹo sử dụng

- **Tích hợp**: Kết hợp S3 với CloudFront để tối ưu phân phối nội dung.
- **Tiết kiệm chi phí**: Dùng Glacier cho dữ liệu ít truy cập, Trusted Advisor để tối ưu hóa.
- **Bảo mật**: Kích hoạt IAM và WAF cho mọi ứng dụng.
- **Tự động hóa**: Sử dụng CloudFormation và CodePipeline cho DevOps.
