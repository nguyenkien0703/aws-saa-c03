## Content

## Luồng làm việc (Workflow)

### Bắt đầu một kho lưu trữ mới (Start a new repository)

| Command | Mô tả |
| --- | --- |
| `git init` | Tạo một kho lưu trữ mới trong thư mục hiện tại |
| `git clone [url] [dir]` | Sao chép một kho lưu trữ từ xa |

## Lịch sử (History)

| Command | Mô tả |
| --- | --- |
| `git log` | Hiển thị lịch sử commit |
| `git log --oneline` | Hiển thị lịch sử commit dạng ngắn gọn |
| `git show` | Hiển thị chi tiết commit gần nhất |
| `git shortlog` | Tóm tắt lịch sử commit theo tác giả |
| `git blame [file]` | Hiển thị ai đã chỉnh sửa từng dòng trong tệp |

## Thay đổi cục bộ (Local changes)

| Command | Mô tả |
| --- | --- |
| `git status` | Kiểm tra trạng thái thay đổi |
| `git add [file]` | Thêm tệp vào vùng chuẩn bị (staging) |
| `git add -p` | Thêm từng phần thay đổi vào staging |
| `git diff` | Xem thay đổi chưa thêm vào staging |
| `git diff --staged` | Xem thay đổi đã thêm vào staging |
| `git commit` | Lưu thay đổi vào kho lưu trữ với thông điệp |
| `git commit -m [message]` | Lưu thay đổi với thông điệp ngắn |
| `git commit --amend` | Sửa đổi commit gần nhất |
| `git reset HEAD [file]` | Xóa tệp khỏi staging nhưng giữ thay đổi |
| `git checkout -- [file]` | Hủy thay đổi trên tệp về trạng thái trước |

## Quản lý nhánh (Branches)

| Command | Mô tả |
| --- | --- |
| `git branch` | Liệt kê các nhánh cục bộ |
| `git branch -r` | Liệt kê các nhánh từ xa |
| `git branch [branch]` | Tạo một nhánh mới |
| `git checkout [branch]` | Chuyển sang nhánh khác |
| `git checkout -b [branch]` | Tạo và chuyển sang nhánh mới |
| `git merge [branch]` | Hợp nhất nhánh vào nhánh hiện tại |
| `git branch -d [branch]` | Xóa một nhánh |
